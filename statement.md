# Problem Statement
1. Users need a simple way to store and manage notes or tasks.
2. Existing digital note apps are complex or require internet access.
3. A lightweight, offline system is needed for quick note creation and retrieval.
4. Beginners need a project that demonstrates basic programming and file handling

# Scope of the Project
1. Allows users to create, view, edit, delete, and search notes.
2. Stores notes permanently using JSON files.
3. Works on any system with Python installed.
4. Designed for academic use by students learning programming.
5. Focuses on basic CRUD operations and simple data storage.

# Target Users
1. Students who want to manage study notes or reminders.
2. Beginners learning Python, functions, and file handling.
3. Users who want a simple offline note-taking tool.
4. Anyone needing a lightweight text-based personal organizer.
   
# High-Level Features
1. Create and save notes with title, content, category, and priority.
2. View all saved notes in a simple list.
3. Edit and update existing notes.
4. Delete unwanted notes.
5. Search notes by keyword or category.
6. Save and load notes automatically using JSON.
7. Clean and easy menu-driven interface.
